## English

Personal portfolio built with HTML, CSS and Bootstrap 5, created as a first-year Web Application Development (DAW) student. It's a responsive single-page website that brings together, in one place, who I am, my front-end and back-end skills, my projects, my education and my work experience.

### Live demo

> `https://portfoliobootstrapmario.netlify.app`

### Built with

- HTML5
- CSS3 (custom styles in `styles.css`)
- [Bootstrap 5](https://getbootstrap.com/) (via CDN)
- Vanilla JavaScript (contact form)

### Project structure

```
.
├── index.html      # Main page (all sections)
├── styles.css      # Custom styles
├── contact.js      # Contact form logic
└── images/         # Profile photo, project screenshots and logos
```

### Sections

- **Home** — intro with profile photo
- **About** — short bio
- **Skills** — front-end and back-end skills with progress bars and tools
- **Projects** — selected web projects
- **Education** — academic background
- **Experience** — work experience
- **Contact** — contact form and social links
